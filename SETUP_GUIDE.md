# セットアップガイド - メルカリ出品アシスタント

## 🚀 5分でインストール完了！

### ステップ1: ファイルの準備

1. `mercari-auto-tool` フォルダ全体をダウンロード
2. 解凍して任意の場所に保存（例: デスクトップ）

### ステップ2: アイコン画像の準備

**重要**: Chrome拡張機能にはアイコンが必須です。

#### オプションA: 簡単な方法（推奨）
以下のサイトでアイコンを生成:
1. [Favicon.io](https://favicon.io/favicon-generator/) にアクセス
2. テキスト: "M" (Mercariの頭文字)
3. 背景色: `#4ECDC4` (ティールブルー)
4. フォント色: `#FFFFFF` (白)
5. ダウンロードして以下をリネーム:
   - `favicon-16x16.png` → `icon16.png`
   - `favicon-32x32.png` → `icon48.png`
   - `android-chrome-192x192.png` → `icon128.png`
6. `icons/` フォルダに配置

#### オプションB: 自分でデザイン
- Canva、Figma、Photoshopなどで作成
- サイズ: 16×16, 48×48, 128×128 px
- PNG形式
- テーマカラー: `#4ECDC4` (ティールブルー)

#### オプションC: 既存の画像を使用
任意の画像を使用する場合:

```bash
# ImageMagickがインストールされている場合
convert your-image.png -resize 16x16 icons/icon16.png
convert your-image.png -resize 48x48 icons/icon48.png
convert your-image.png -resize 128x128 icons/icon128.png
```

### ステップ3: Chrome拡張機能としてインストール

1. **Chromeを開く**
   - アドレスバーに `chrome://extensions/` と入力してEnter

2. **デベロッパーモードをON**
   - 右上の「デベロッパーモード」トグルをクリック

3. **拡張機能を読み込む**
   - 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
   - `mercari-auto-tool` フォルダを選択
   - 「フォルダーの選択」をクリック

4. **完了！**
   - 拡張機能一覧に「メルカリ出品アシスタント」が表示されます
   - ピンアイコンをクリックしてツールバーに固定（推奨）

### ステップ4: 動作確認

1. **メルカリにアクセス**
   ```
   https://jp.mercari.com/
   ```

2. **拡張機能を開く**
   - ツールバーのアイコンをクリック
   - ポップアップが表示されればOK！

3. **デモ機能を試す**
   - 「100円値下げ実行」ボタンをクリック
   - 「出品分析」ボタンをクリック
   - 自動値下げをON/OFF

## ⚙️ 設定のカスタマイズ

### 自動値下げのスケジュール

1. ポップアップを開く
2. 「自動値下げ設定」セクションを見つける
3. トグルをON
4. 実行時間を選択（例: 深夜0時）
5. 最低価格を設定（例: 500円）

### 推奨設定

```
実行時間: 深夜0時または朝6時
最低価格: 500円〜1,000円
自動値下げ: ONにする前に必ず手動テスト
```

## 🐛 トラブルシューティング

### エラー: 「拡張機能を読み込めませんでした」

**原因**: アイコンファイルが見つからない

**解決方法**:
1. `icons/` フォルダに以下が存在するか確認:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`
2. ファイル名が正確か確認（大文字小文字も含む）
3. PNG形式であることを確認

### エラー: 「このページでは動作しません」

**原因**: メルカリ以外のページで実行

**解決方法**:
- `https://jp.mercari.com/` を開いてから実行
- 特に `/mypage/listings` （出品一覧）で実行

### ポップアップが真っ白

**原因**: JavaScriptエラー

**解決方法**:
1. `chrome://extensions/` を開く
2. 拡張機能の「エラー」ボタンをクリック
3. エラーメッセージを確認
4. Chrome DevToolsを開く（F12）
5. Consoleタブでエラーログ確認

### 値下げが実行されない

**現在の状態**: デモ版

このプロトタイプは実際の値下げは行いません。
- ログ出力のみ（Chrome DevTools Console確認）
- 実装版では実際の操作を行います

## 🔧 開発者向け: コードの編集

### ファイルを編集した場合

1. ファイルを保存
2. `chrome://extensions/` を開く
3. 拡張機能の「更新」ボタンをクリック（リロードアイコン）
4. ページをリロード（F5）

### デバッグ方法

#### ポップアップのデバッグ
1. 拡張機能アイコンを右クリック
2. 「ポップアップを検証」を選択
3. DevToolsが開きます

#### Content Scriptのデバッグ
1. メルカリページでF12を押す
2. Consoleタブでログ確認
3. Sourcesタブで `content.js` を確認

#### Background Scriptのデバッグ
1. `chrome://extensions/` を開く
2. 拡張機能の「service worker」をクリック
3. DevToolsが開きます

```css
}
```
