# メルカリ出品アシスタント - Chrome Extension

メルカリの出品管理を効率化するChrome拡張機能のプロトタイプです。

UI UX Pro Max skillを活用して構築:
- **スタイル**: Minimalism + Glassmorphism
- **カラーパレット**: Fintech系（信頼感のある配色）
- **フォント**: Inter + システムフォント
- **レイアウト**: Bento Grid + カード型デザイン

## 📦 インストール方法

### 開発版（ローカル）

1. このフォルダをダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

## 🚀 使い方

1. 拡張機能アイコンをクリックしてポップアップを開く
2. メルカリの出品一覧ページを開く
3. 「100円値下げ実行」ボタンで値下げ
4. 「出品分析」で統計情報を取得
5. 自動値下げをONにしてスケジュール設定

## 🛠️ 技術スタック

- **Manifest V3** - 最新のChrome Extension仕様
- **Vanilla JavaScript** - フレームワーク不要で軽量
- **CSS3** - UI UX Pro Max準拠のデザイン
- **Chrome Storage API** - 設定の永続化
- **Chrome Alarms API** - スケジューリング

## ⚠️ 重要な注意事項

### 規約について
このプロトタイプは**教育目的**で作成されています。実際に使用する場合は以下に注意:

1. **メルカリの利用規約を遵守する**
   - 自動化ツールの使用に関する規約を確認
   - 過度なリクエストでサーバーに負荷をかけない
   
2. **レート制限を守る**
   - 1秒に1回程度のリクエストに制限
   - 短時間での大量操作は避ける

3. **ユーザー操作の補助として設計**
   - 完全自動化ではなく、ユーザー確認を挟む
   - ブラウザ上での操作を補助する形

### デモ版の制限
現在のバージョンは以下の制限があります:

- ✗ 実際のメルカリAPIには接続していません
- ✗ 実際の値下げ処理は行いません
- ✗ ダミーデータを表示しています
