# ✅ インストールチェックリスト

プロトタイプを正しくインストールするためのチェックリスト

## 事前準備

- [ ] Google Chrome がインストールされている（最新版推奨）
- [ ] `mercari-auto-tool` フォルダをダウンロード済み
- [ ] フォルダの場所を把握している（デスクトップなど）

## ファイル確認

以下のファイルがすべて存在することを確認:

### 必須ファイル
- [ ] `manifest.json` - 拡張機能の設定
- [ ] `popup.html` - UIのHTML
- [ ] `popup.css` - UIのスタイル
- [ ] `popup.js` - UIのロジック
- [ ] `content.js` - メルカリページ用スクリプト
- [ ] `background.js` - バックグラウンド処理
- [ ] `content.css` - ページ用スタイル

### アイコンファイル
- [ ] `icons/icon16.png` (16×16px)
- [ ] `icons/icon48.png` (48×48px)
- [ ] `icons/icon128.png` (128×128px)

### ドキュメント
- [ ] `README.md`
- [ ] `SETUP_GUIDE.md`
- [ ] `QUICKSTART.md`

## インストール手順

### 1. Chrome拡張機能ページを開く
- [ ] `chrome://extensions/` にアクセス
- [ ] ページが正しく開いた

### 2. デベロッパーモードを有効化
- [ ] 右上の「デベロッパーモード」トグルをクリック
- [ ] トグルが青色（ON状態）になった

### 3. 拡張機能を読み込む
- [ ] 「パッケージ化されていない拡張機能を読み込む」ボタンが表示された
- [ ] ボタンをクリック
- [ ] ファイル選択ダイアログが開いた
- [ ] `mercari-auto-tool` フォルダを選択
- [ ] 「フォルダーの選択」をクリック

### 4. インストール確認
- [ ] 拡張機能一覧に「メルカリ出品アシスタント」が表示された
- [ ] エラーが表示されていない
- [ ] アイコンが正しく表示されている

### 5. ツールバーに固定（推奨）
- [ ] 拡張機能一覧の右側にあるピンアイコンをクリック
- [ ] Chromeツールバーにアイコンが表示された

## 動作確認

### 基本動作
- [ ] ツールバーのアイコンをクリック
- [ ] ポップアップが開いた
- [ ] UIが正しく表示されている
- [ ] 統計情報（0件、¥0など）が表示されている

### メルカリページでの動作
- [ ] `https://jp.mercari.com/` を開く
- [ ] ページが正常に表示される
- [ ] 拡張機能アイコンをクリック
- [ ] ポップアップが開く

### 機能テスト
- [ ] 「100円値下げ実行」ボタンをクリックできる
- [ ] 「出品分析」ボタンをクリックできる
- [ ] 自動値下げトグルをON/OFFできる
- [ ] スケジュール時間を変更できる
- [ ] 最低価格を入力できる

## トラブルシューティング

### エラーが表示される場合

#### "拡張機能を読み込めませんでした"
- [ ] `icons/` フォルダに3つの画像があるか確認
- [ ] ファイル名が正確か確認（icon16.png, icon48.png, icon128.png）
- [ ] PNG形式であることを確認

#### ポップアップが真っ白
- [ ] Chrome DevTools を開く（F12）
- [ ] Consoleタブでエラーメッセージを確認
- [ ] エラーメッセージをメモ

#### 「このページでは動作しません」
- [ ] メルカリのページ（jp.mercari.com）を開いているか確認
- [ ] ページをリロード（F5）
- [ ] 拡張機能を再読み込み

### 解決しない場合
- [ ] Chrome を再起動
- [ ] 拡張機能を削除して再インストール
- [ ] 別のChromeプロファイルで試す

## デバッグモード

開発者向けのデバッグ確認:

### Console ログ確認
- [ ] ポップアップを右クリック → 「検証」
- [ ] Console に "メルカリアシスタント起動" と表示
- [ ] エラーが表示されていない

### Background Script 確認
- [ ] `chrome://extensions/` を開く
- [ ] 拡張機能の「service worker」をクリック
- [ ] DevTools が開く
- [ ] Console に起動メッセージが表示

### Content Script 確認
- [ ] メルカリページで F12 を押す
- [ ] Console タブを確認
- [ ] "メルカリアシスタント Content Script 起動" と表示

## 完了確認

すべてチェックが完了したら:

- [ ] インストールが正常に完了した
- [ ] 基本機能が動作している
- [ ] エラーが発生していない
